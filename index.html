GARantir

A Smart Safe Concept. Personalised Compartments and Authentication.

Garantir is a smart safe concept that enables one safe to be used by many and with individual compartments. Every user has a profile and compartment allocated, only through pin code or card access can the user access their compartment, making it secure from shared users. The safe is also connected to  the internet, which provides live updates on whether the safe is enabled or disabled. A built in camera would take a  picture after 5 incorrect attempts, and puts it into disabled mode that only the app connected can unlock.

  • Date: October 2017 - February 2018
  • Client: Ngee Ann Polytechnic
  • Category: Internet of Things

P-LOT

Parking Booking System

Finding a Parking Lot has always been a driver's headache when visiting anywhere. This project aims to eleviate that problem. With a web application, drivers are able to make bookings of specific lots allocated at the higher levels of any mainstream multi-storey carpark. The physical lot would have an ultrasonic sensor that would detect if the car is there and an lcd display to display the car's license plate number upon booking. After parking, users would have to scan a QR code sent to them upon booking to authenticate and verify the user's identity. if not, an alarm would ring after five minutes if not verified. These special lots can also be disabled by an admin if there were to be a special event for that venue to enable free parking.

  • Date: October 2018 - February 2019
  • Client: Land Transport Authority
  • Category: Transport
